# Usage

## Scope
系统状态查询与基础系统操作。

## Tools
- check_process_status
- list_processes
- show_desktop
- get_current_time
- take_screenshot
- open_notepad
- read_notepad_text
- add_operation_experience
- get_operation_experience
- delete_image
- compress_operation_experience
- create_task_plan
- read_task_plan
- mark_task_completed
- append_task_step

## Examples
- 先检查进程再决定是否启动软件
- 截图用于 OCR 或视觉理解
- 读取记事本内容进行后续分析
- 记录系统操作经验到向量库 (RAG)
- 语义检索过往经验辅助决策
- 删除截图等临时图片
- 创建任务计划拆解复杂任务 (Step-by-Step Plan)
- 标记步骤完成并自动读取下一步

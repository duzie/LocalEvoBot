# Usage

## Scope
浏览器自动化操作与基础网页测试执行。

## Tools
- selenium_open_url
- selenium_click
- selenium_type
- selenium_get_text
- selenium_wait_for
- selenium_assert_text
- selenium_assert_title
- selenium_take_screenshot
- selenium_run_steps
- selenium_close

## Examples
- 打开网页后输入关键词并搜索
- 点击页面按钮触发操作
- 读取页面关键文本
- 按步骤执行测试并输出报告

# Skill

## Name
selenium_skill

## Version
1.0.0

## Description
基于 Selenium 的浏览器自动化与基础测试执行。

## Entry
app.skills.selenium_skill.scripts

## Tools
- selenium_open_url: 打开网页并保持会话
- selenium_click: 点击页面元素
- selenium_type: 输入文本到页面元素
- selenium_get_text: 获取元素文本
- selenium_wait_for: 等待元素满足条件
- selenium_assert_text: 断言元素文本
- selenium_assert_title: 断言页面标题
- selenium_take_screenshot: 保存页面截图
- selenium_run_steps: 执行测试步骤并输出报告
- selenium_close: 关闭浏览器会话

## Platforms
- Windows

## Dependencies
- selenium
